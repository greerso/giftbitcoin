## Crypto constants (do not rename for brand)

Protocol labels stay fixed for address compatibility (SPEC v1):

- NUMS seed: `BTCGiftcard/v1/NUMS`  
- HKDF info: `btcgiftcard/v1/claim`, `btcgiftcard/v1/refund`  
- Scalar label: `btcgiftcard/v1/scalar`  

Changing these would break golden vectors and existing gift packages.
